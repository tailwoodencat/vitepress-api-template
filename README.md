## vitepress-api-template

## features

- `vitepress` 1.6.3+ [https://vitepress.dev/](https://vitepress.dev/)
- management by `pnpm`
- let `docs` folder to be the root

## env

- node `>=20.17.0`
- pnpm `10.10.0`
- build out folder is `docs/.vitepress/dist`
